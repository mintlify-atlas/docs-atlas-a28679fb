# Quick Start Guide

Get up and running with Zenfi SDK in just a few steps.

## Prerequisites

Before you begin, make sure you have:

- Your partner name provided by Zenfi
- A domain where you'll be using the SDK
- Basic knowledge of JavaScript

## Basic Setup

<Steps>
  <Step title="Install the SDK">
    Install via npm or add the CDN script to your HTML:
    
    ```bash
    npm install --save @zenfi/js-sdk
    ```
    
    See the [Installation](/installation) page for detailed instructions.
  </Step>
  
  <Step title="Initialize the SDK">
    Create a new instance of ZenfiSDK with your partner configuration:
    
    ```javascript
    const ZenfiSDK = require('@zenfi/js-sdk');
    
    const zenfi = new ZenfiSDK({
      partnerName: 'credifast',
      cookiesDomain: 'credifast.com',
      targets: [
        {
          dataKey: 'name',
          strategy: 'value',
          selector: '#nameInput'
        },
        {
          dataKey: 'email',
          strategy: 'value',
          selector: '#emailInput'
        }
      ]
    });
    ```
  </Step>
  
  <Step title="Auto-fill form fields">
    Call `fillTargets()` to automatically populate form fields with user data:
    
    ```javascript
    await zenfi.fillTargets();
    // Form inputs will be automatically filled with lead information
    ```
    
    This will:
    - Fetch lead information from the Zenfi API
    - Auto-fill the configured target elements with user data
  </Step>
  
  <Step title="Track events">
    Track user actions throughout your funnel:
    
    ```javascript
    // Track a conversion event
    zenfi.trackConversion('addToCart', {
      price: 100,
      amount: 2
    });
    
    // Track page views automatically
    zenfi.initPageViewsTracking();
    ```
  </Step>
</Steps>

## Complete Example

Here's a complete example showing form auto-fill with advanced features:

```javascript
const zenfi = new ZenfiSDK({
  partnerName: 'credifast',
  cookiesDomain: 'credifast.com',
  targets: [
    {
      dataKey: 'name',
      strategy: 'value',
      selector: '#nameInput'
    },
    {
      dataKey: 'email',
      strategy: 'value',
      selector: ({ dataKey }) => `#${dataKey}Input`,
      beforeAction: () => console.log('before filling info'),
      afterAction: ({ element }) => element.classList.add('changed')
    }
  ]
});

// Auto-fill the form
await zenfi.fillTargets();

// Track page views automatically
zenfi.initPageViewsTracking();

// Track custom conversion events
zenfi.trackConversion('formSubmitted', {
  formType: 'registration',
  timestamp: new Date().toISOString()
});
```

## Fetching Lead Data

You can also fetch lead information without auto-filling:

```javascript
const zenfi = new ZenfiSDK();
await zenfi.fetchData();

console.log(zenfi.leadInfo);
/*
Prints:
{
  name: 'Chewbacca',
  email: 'chewy@lucasfilm.com'
}
*/
```

<Note>
  The SDK automatically handles lead identification through URL parameters and cookies. Make sure users arrive at your site with the proper Zenfi tracking parameters.
</Note>

## Configuration Options

When creating a ZenfiSDK instance, you can configure:

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `partnerName` | String | Yes (for events) | Your partner name provided by Zenfi |
| `cookiesDomain` | String | No | Domain where cookies will be available |
| `targets` | Array | No | Configuration for auto-filling form elements |

### Target Configuration

Each target in the `targets` array supports:

| Property | Type | Default | Description |
| -------- | ---- | ------- | ----------- |
| `dataKey` | String | Required | Key name in the lead data object |
| `selector` | String or Function | Required | CSS selector or function returning selector |
| `strategy` | `'value'` \| `'text'` \| `'click'` | `'value'` | How to fill the element |
| `beforeAction` | Function | Optional | Callback before filling |
| `afterAction` | Function | Optional | Callback after filling |

## Tracking Events

The SDK provides multiple methods for event tracking:

<CodeGroup>

```javascript Conversion Events
// Track when users move forward in the funnel
zenfi.trackConversion('register', {
  source: 'homepage',
  userId: '12345'
});
```

```javascript Abortion Events
// Track when users are rejected or leave
zenfi.trackAbortion('declinedOffer', {
  reason: 'User got a better interest rate elsewhere'
});
```

```javascript Page Views
// Track a single page view
zenfi.trackPageView();

// Or enable automatic tracking
zenfi.initPageViewsTracking();
```

</CodeGroup>
