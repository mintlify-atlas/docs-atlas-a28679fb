# Zenfi JavaScript SDK

The Zenfi JavaScript SDK is a powerful tool for Zenfi partners to streamline user experience and track engagement in their applications.

## What is Zenfi SDK?

Zenfi SDK is a lightweight JavaScript library that enables you to:

- **Auto-fill user data** - Automatically populate form inputs with lead information, reducing friction in the user journey
- **Track conversions** - Monitor when users move forward in your funnel (registrations, clicks, purchases)
- **Track abandonment** - Capture when users are rejected or leave the conversion flow
- **Monitor page views** - Automatically or manually track user navigation across your application

## How It Works

The SDK works by:

1. **Capturing lead information** from URL parameters or cookies
2. **Fetching user data** from the Zenfi API using secure tokens
3. **Auto-filling forms** based on your target configuration
4. **Sending events** to Zenfi when users convert or abandon the flow

## Use Cases

- **Financial applications** - Pre-fill loan applications with verified user data
- **E-commerce platforms** - Streamline checkout by auto-completing customer information
- **Lead generation** - Track user progression through multi-step forms
- **Partner integrations** - Seamlessly pass user data between Zenfi and partner platforms
